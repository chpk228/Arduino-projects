Arduino is an open-source electronics platform used to build digital devices that can sense and control things in the physical world.
Arduino is a small computer you can program to do things.

A USB port (for programming)

A microcontroller (brain of the board)

Digital pins (on/off signals)

Analog pins (read sensor data)

Power pins (5V, GND)

An onboard LED on pin 13.
An Arduino board (like Uno, Nano, etc.)

A USB cable

A computer with the Arduino IDE (software)
